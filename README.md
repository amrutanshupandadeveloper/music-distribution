# music-distribution
music-app-updates
🎵 Music App - Updates & Distribution 🚀
Welcome to the official distribution repository for the Music App. This repository is used to host the latest releases, update metadata, and provide a direct download link for users.
✨ Features of the App

🎧 Unlimited Streaming: Access millions of songs online for free.

📂 Local Playback: Play all your high-quality offline music files.

📜 Synced Lyrics: Real-time lyrics for your favorite tracks (LrcLib & Paxsenix).

🎨 Dynamic Themes: The app UI changes colors based on the song's album art.

⏬ Fast Downloads: Download songs directly to your device with metadata embedding.

📱 Modern UI: Built with Jetpack Compose for a smooth and fluid experience.

🔔 Background Playback: Fully supported with Media3 and Notification controls.
📥 How to Install?
1.Go to the  section.
2.Download the latest .apk file from the Assets.
3.Open the file on your Android device and install it.
◦Note: You might need to enable "Install from Unknown Sources" in your settings.
🛠️ Requirements
•Android Version: Android 8.0 (API 26) or higher.
•Permissions: Needs Media and Notification access for the full experience.
🔄 Automatic Updates
The app automatically checks this repository for new updates every time you launch it. No need to check back here manually!
